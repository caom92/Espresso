Esta aplicacion requiere de PHPMailer, la cual requiere composer y 
php 5.6+

Para instalar phpmailer, visita este link:
https://github.com/PHPMailer/PHPMailer

Es mejor utilizar el sistema OAUTH2 de smtp.google.com para hacer el 
envio de reportes de error por correo electronico, las instrucciones
para instalarlo estan aqui:
https://github.com/PHPMailer/PHPMailer/wiki/Using-Gmail-with-XOAUTH2

Finalmente, cualquier problema que tengas, revisa aqui:
https://github.com/PHPMailer/PHPMailer/wiki/Troubleshooting

Todo lo que necesitas saber para poner a punto el sistema de PHPMailer esta
en esos 3 links.
