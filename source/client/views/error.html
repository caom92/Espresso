error
Parece que hubo un problema.

  • help_circle ¿Qué ocurrió?

    Lo sentimos, pero hubo un problema al procesar una operación en la base de datos del servidor; el sistema intentó consultar o modificar el contenido de la base de datos y no pudo completar esta acción correctamente.

  • help_circle ¿Qué lo causó?

    Este tipo de errores son poco comunes, pues normalmente son ocasionados por defectos técnicos en el lado del servidor y no por un descuido por parte del usuario. Las causas más comunes son:

    • No se pudo establecer una conexión adecuada con la base de datos del servidor, ya sea porque no hay conexión a Internet o porque el servidor se encuentra apagado o deshabilitado.
    • No se cuentan con los permisos adecuados para acceder a la base de datos del servidor. Esto se debe normalmente a una mala configuración del servidor o de la base de datos.
    • Se intentó consultar una tabla que no existe en la base de datos del servidor. Esto indica un error en la programación del sistema o en la configuración de la base de datos.
  • help_circle ¿Qué puedo hacer para resolverlo?

    Como este tipo de problemas son causados por defectos técnicos sobre los cuales el usuario normalmente no tiene control alguno, hay realmente muy poco que pueda hacer para resolverlo sin la ayuda de un técnico.

    • Revise que su computadora esté conectada a Internet. Si no lo está, reinicie su módem o contacte a su proveedor.
    • Antes de iniciar sesión, le sugerimos que revise atentamente el ícono que indica el estado del servidor. Si está rojo, quiere decir que el servidor no está disponible y le sugerimos contactar al técnico encargado.
